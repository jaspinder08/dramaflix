# 📺 DramaFlix

DramaFlix is a premium mobile OTT (Over-The-Top) streaming application designed for a seamless drama viewing experience. Built with Flutter and Supabase, it features a modern UI, robust authentication, and high-performance video streaming.

## 🚀 Features

- **Personalized Home Feed**: Curated content and featured banners.
- **Story Exploration**: Deep dive into story details and episodes.
- **High-Quality Player**: Smooth video playback with "Continue Watching" support.
- **Secure Auth**: Authentication powered by Supabase and Riverpod.
- **Admin Dashboard**: Comprehensive management tools for content creators.

## 🛠 Tech Stack

- **Frontend**: Flutter (Riverpod for State Management)
- **Backend**: Supabase (Auth, Database, Storage)
- **Navigation**: GoRouter
- **HTTP Client**: Dio
- **Animations**: Flutter Animate

## 📂 Project Structure

- `dramaflix_user/`: The primary Flutter application for end-users.
- `dramaflix_admin/`: Administrative dashboard for managing stories and content.

## 🚦 Getting Started

### Prerequisites
- Flutter SDK (latest stable)
- Dart SDK
- A Supabase project

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jaspinder08/dramaflix.git
   ```

2. Navigate to the user app directory:
   ```bash
   cd dramaflix_user
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
